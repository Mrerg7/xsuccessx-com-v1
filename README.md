# xsuccessx.com

Static Astro site for **xsuccessx.com** — an elite X.com creator guide with domain acquisition CTA.

## Stack

- [Astro](https://astro.build) 6 (pure static output, no adapter)
- Tailwind CSS 4
- Content Collections (creator genres, pro tips, platform features)
- Cloudflare Workers Static Assets (`wrangler.toml`)
- Cloudflare Images CDN for hero and section graphics

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run deploy   # builds then wrangler deploy
```
